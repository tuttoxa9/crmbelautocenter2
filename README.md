# Белавтоцентр CRM

Внутренняя CRM-система для менеджеров автохауса «Белавтоцентр» (belautocenter.by). 

## О проекте
Данный проект представляет собой изолированное внутреннее приложение для управления лидами (заявками с сайта, TikTok, Instagram, входящими звонками) и ведения учета клиентов. В будущем планируется интеграция с Cloudflare R2 для управления фотографиями автомобилей. Приложение использует единую базу данных Firebase Firestore и аутентификацию Firebase Auth с основным сайтом.

## Технологический стек
- **Framework:** Next.js 14+ (App Router)
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS
- **UI Компоненты:** shadcn/ui
- **База данных & Авторизация:** Firebase (Firestore & Firebase Auth)
- **Иконки:** Lucide React
- **Хостинг:** Vercel

## Структура проекта
- `src/app`: Страницы и роутинг (Next.js App Router).
- `src/components/layout`: Переиспользуемые компоненты макета (Sidebar, DashboardLayout, AuthGuard).
- `src/components/ui`: UI-компоненты на базе shadcn/ui.
- `src/contexts`: React-контексты (AuthContext для управления состоянием пользователя).
- `src/lib`: Утилиты и конфигурация Firebase (`firebase.ts`).

## Запуск локально

1. Склонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd belautocenter-crm
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Настройте переменные окружения:
   Скопируйте `.env.example` в `.env.local` и заполните значения для Firebase:
   ```bash
   cp .env.example .env.local
   ```
   Вам понадобятся ключи от вашего Firebase проекта.

4. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

5. Откройте [http://localhost:3000](http://localhost:3000) в браузере. Вы будете перенаправлены на страницу авторизации `/login`.

## Дизайн и стилизация
Интерфейс оформлен в строгом, чистом, профессиональном стиле с использованием тёмной боковой панели навигации и светлой основной области контента. Приложение полностью адаптивно для мобильных устройств.
